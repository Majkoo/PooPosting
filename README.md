## Tech Stack

**Client:** Angular, Tailwind

**Server:** .NET

**Database:** Supabase


## Run Locally

Clone the project

```bash
  git clone https://github.com/PooPosting-Monorepo/PooPosting
```

Go to the project directory

```bash
  cd WebClient
```

Install dependencies

```bash
  npm ci
```

Start the server

```bash
  ng serve
```


## Deployment

To deploy this project, open /WebClient and run

```bash
  ng serve
```
